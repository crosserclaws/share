## Homework

### Description

实现一个打印图形面积的函数，它接收一个可以计算面积的类型作为参数，比如圆形，三角形，正方形，需要用到泛型和泛型约束

### Result

```shell
$ cargo run
The area of shape Square { edge: 2.0 } is 4
The area of shape Circle { radius: 2.0 } is 12.566370614359172
```
